# Satis-Performansi-Bazar-Davranisi-Analizi
Dost Tədris Mərkəzində keçdiyim təlim müddətində Power BI proqramından istifadə edərək satış məlumatları əsasında interaktiv dashboard hazırlamaq imkanı əldə etdim. Təlim çərçivəsində təqdim olunan Excel faylı üzərində işləyərək satış, xərc, gəlir, məhsul performansı və ölkələr üzrə satış göstəricilərini analiz etdim.
Hesabatda təhlil etdiyim əsas göstəricilər:

◾ Ümumi satış məbləği, ümumi xərclər, ümumi gəlir və satılan məhsul miqdarı

◾ Həftə içi və həftə sonu satışların payı və müqayisəsi

◾ Həftə günləri üzrə satışların faiz bölgüsü

◾ İllər üzrə xərclərin dəyişməsi və artım dinamikası

◾ Müştərilər üzrə həftə içi və həftə sonu satış payları

◾ Aylar üzrə ümumi satış məbləği və satılan məhsul miqdarı

◾ Ölkələr üzrə cari və keçən ilin satış göstəricilərinin müqayisəsi

◾ Məhsullar üzrə keçən ilə nisbətən satış fərqi və performans göstəriciləri
